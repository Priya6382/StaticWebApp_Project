19MAM58 - Cloud Computing Lab Project

Project - Title

Our Project Is

  1. Create a Dockerfile with the necessary instructions for building the image.
  2. Build the Docker image using the Dockerfile.
  3. Tag the Docker image for pushing it to a container registry (e.g., Docker Hub).
  4. Push the Docker image to the container registry.
  5. Install Minikube for local Kubernetes development.
  6. Start Minikube to create a local Kubernetes cluster.
  7. Deploy the Docker image to the Minikube cluster.
  8. Access and test the deployed application within the Minikube cluster.
